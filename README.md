# MarketWave Media

# What is MarketWaves Media?

Marketwaves media is a digital marketing or social media marketing agency, that I was developing with some
of my friends😁, and we decided to build ourselves a website.

Basically our agency helps clients with all of their digital marketing needs from social media marketing to branding and media management.

# Tech Stack
This was my first professional website which I collaborated with one of my friend on. It was built using just HTML, CSS, JS, and Tailwind.
This explains some of the rough edges as we were still improving.
